## 🚀 التشغيل السريع

```bash
# استنساخ المشروع
git clone https://github.com/username/alshamel.git
cd alshamel

# تثبيت التبعيات
bun install

# إعداد قاعدة البيانات
bun run db:push

# تشغيل خادم التطوير
bun run dev
```

الموقع سيكون متاحاً على `http://localhost:3000`

## 🛠️ التقنيات المستخدمة

### Frontend
- **Next.js 16** - إطار React للإنتاج
- **React 19** - مكتبة واجهة المستخدم
- **TypeScript** - لغة برمجة مُنوعة
- **Tailwind CSS** - إطار CSS
- **shadcn/ui** - مكونات UI

### Backend
- **Prisma ORM** - للتعامل مع قاعدة البيانات
- **SQLite** - قاعدة بيانات مدمجة
- **API Routes** - واجهات برمجة التطبيقات

### المكتبات
- **Framer Motion** - الحركات والانتقالات
- **Lucide React** - الأيقونات
- **next-themes** - الوضع الداكن
