## 📝 وصف المشروع

**المحاسب الشامل** هو موقع تسويقي متكامل لبرنامج محاسبي سحابي سعودي متوافق مع هيئة الزكاة والضريبة والجمارك (ZATCA). الموقع يقدم:

- عرض المنتجات المحاسبية (6 منتجات)
- عرض الموديولات المتخصصة (16+ موديول)
- عرض القطاعات التجارية المدعومة (12 قطاع)
- مدونة مقالات محاسبية
- نظام تواصل وطلب عروض تجريبية
- دعم الفوترة الإلكترونية

## 🎯 الهدف من المشروع

### الأهداف الرئيسية:
1. **التسويق الرقمي** - تسويق منتجات المحاسب الشامل عبر الويب
2. **جمع العملاء المحتملين** - عبر نماذج التواصل وطلب العروض
3. **SEO محسن** - ظهور في نتائج البحث الأولى
4. **تجربة مستخدم متميزة** - واجهة عربية سلسة مع وضع داكن

### الجمهور المستهدف:
- أصحاب الأعمال في السعودية والخليج
- المحاسبون والمدراء الماليون
- رواد الأعمال والشركات الناشئة
- المنشآت التي تحتاج امتثال ZATCA
